一、目录操作

📌 基础命令

命令 作用
ls 列出目录内容
cd 切换目录
pwd 显示当前路径
mkdir 创建目录
rmdir 删除空目录
cp 复制文件/目录
rm 删除文件/目录
mv 移动/重命名

📌 目录切换 (cd) 

cd /      # 进入根目录  
cd ~      # 进入用户主目录  
cd -      # 返回上次访问的目录  
cd ..     # 返回上一级目录

📌 查看目录 (ls & pwd) 

ls -a  # 显示所有文件（含隐藏）  
ls -l  # 详细列表（等同于 ll）  
pwd    # 显示当前路径

📌 目录管理

创建 (mkdir)

mkdir data/             # 创建 data 目录  
mkdir -p hadoop/data/   # 递归创建多级目录

删除 (rm & rmdir)

rmdir bigdata      # 删除空目录  
rm -rf data/       # 强制删除目录及内容  
rm -rf *           # ⚠️ 删除所有文件，慎用！

移动/重命名 (mv & cp)

mv hadoop/ hadoop01/   # 重命名目录  
mv data Data/          # 移动目录  
cp -r apps/ Data/      # 复制目录

查找 (find)

find /tmp -name 'a*'   # 查找以 'a' 开头的文件/目录

二、文件操作

📌 基础管理

创建 (touch)

touch a.txt              # 创建 a.txt  
touch b.txt c.txt d.txt  # 一次创建多个文件

删除 (rm)

rm -rf /tmp/a.txt        # 删除文件

编辑 (vi & vim)

vi a.txt                 # 编辑文件

移动/重命名 (mv)

mv hello.txt hi.txt        # 重命名  
mv /tmp/a.txt /tmp/b.txt   # 移动并重命名

复制 (cp)

cp a.txt hadoop/         # 复制到 hadoop 目录  
cp a.txt hadoop/b.txt    # 复制并重命名

📌 查看文件内容

命令 作用
cat 显示全部内容
more 分页查看
less 可上下翻页查看
head -N 显示前 N 行
tail -N 显示后 N 行 
cat /etc/sudo.conf       # 直接查看文件  
more /etc/sudo.conf      # 分页查看（回车翻行，空格翻页，q 退出）  
less /etc/sudo.conf      # 上下翻页查看（PgUp/PgDn，q 退出）  
head -5 /etc/sudo.conf   # 显示前 5 行  
tail -5 /etc/sudo.conf   # 显示后 5 行

📌 创建链接 (ln) 

ln data.txt d   # 创建硬链接
